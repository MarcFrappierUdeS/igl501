Installation de Prob

Voir https://www3.hhu.de/stups/prob/index.php/Download

ProB utilise le logiciel Tcl/Tk; la page de téléchargement de ProB contient une référence à un site pour télécharger Tcl/Tk, mais ce lien est maintenant invalide.

Pour installer Tcl/Tk, vous pouvez utiliser ce lien:

https://www.activestate.com/products/tcl/downloads/

Ce site vous demande de vous enregistrer; à ma connaissance, c’est un site très fiable; c’est la version que j’ai installée sur mon ordinateur.
